# designpro-api

API de orquestación para DesignPro.

## Responsabilidades
- crear jobs
- consultar estado de jobs
- publicar resultados
- exponer catálogos

## Endpoints objetivo
- `POST /designpro/jobs`
- `GET /designpro/jobs/:id`
- `GET /designpro/jobs/:id/result`
- `GET /designpro/catalogs`
- `GET /health`
